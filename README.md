# Maryland TOD AI Dashboard 🚆

**1st Place — UMD SAC 7th Annual Datathon 2026**

An AI-powered Transit-Oriented Development strategy for Maryland, built on 12 real government datasets.

## 4 AI Features

| Feature | Algorithm |
|---|---|
| Corridor Scoring Engine | K-Means Clustering + PCA + Weighted Scoring |
| Demand Forecasting | Polynomial Regression + Monte Carlo (5,000 runs) |
| Equity Recommender | Content-Based Filtering + Cosine Similarity |
| Conflict Detection | Haversine Spatial Scanning + Rule-Based AI |

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to Streamlit Cloud
